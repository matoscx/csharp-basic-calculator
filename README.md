# csharp-basic-calculator
Calculadora en C# con Windows Forms

Este es un proyecto sencillo de una calculadora de escritorio desarrollada en C# utilizando el entorno de desarrollo SharpDevelop y el framework Windows Forms. Es un ejemplo práctico de cómo manejar eventos y lógica matemática básica en una interfaz gráfica.

El Proyecto

El código se encarga de inicializar la aplicación y cargar el formulario principal (MainForm). Está diseñado siguiendo la estructura estándar de aplicaciones de escritorio en .NET, separando la lógica de inicio de la lógica visual.

Características

Interfaz gráfica intuitiva (Windows Forms).

Manejo de estilos visuales modernos mediante EnableVisualStyles.

Punto de entrada optimizado en el archivo Program.cs.

Tecnologías Utilizadas

Lenguaje: C#

Framework: .NET / Windows Forms

IDE: SharpDevelop (o compatible con Visual Studio)
